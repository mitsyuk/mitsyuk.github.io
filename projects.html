Projects

Alex is involved in several reseach and applied projects. He is most active in the following ones:
Year Project Role
2021- Petri nets with extensions: foundations and applications. Researcher
2020- What software architecture styles are popular? Project Lead, Researcher
2017- Methods for system structure and behaviour visualisation using high-level graph-based models, including Petri nets. Project Lead, Researcher
2013-2019 Gena (450+ downloads): here the goal is to develop methods to simulate process models and generate artificial event logs with suitable properties. Project Lead
2013-2019 Iskra: here the methods to repair process models according to event logs are developed. Project Lead, Developer
2013-2016 Carassius (800+ downloads): this is just a simple petri net (.pnml) editor. Project Lead, Developer
